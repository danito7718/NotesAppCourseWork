Уеб приложение за организиране на бележки
Описание

Настоящото хранилище съдържа курсова работа по дисциплината „Програмиране в Интернет среда“.
Проектът представлява уеб приложение за организиране и управление на бележки, разработено с ASP.NET Core MVC.

Приложението позволява създаване, преглед, редактиране и изтриване на бележки чрез уеб интерфейс.

Използвани технологии

C#

ASP.NET Core MVC

Entity Framework Core

SQLite

ASP.NET Core Identity

HTML и CSS (генерирани изгледи чрез MVC)

Visual Studio

Функционалности

Създаване на бележки

Преглед на списък с бележки

Редактиране на съществуващи бележки

Изтриване на бележки

Съхранение на данните в релационна база (SQLite)

Изпълнете миграциите за базата данни:

Add-Migration InitialCreate
Update-Database


Цел на проекта

Целта на курсовата работа е да демонстрира прилагане на основни принципи на уеб програмирането чрез ASP.NET Core MVC, както и работа с база данни чрез Entity Framework Core.
